## About Phylogenetics Assignment

- Datasets: Compilation of Pinniped FASTA DNA sequences from [Pinnipeds_initdata.txt](https://github.com/itsmichelletrinh/R/blob/main/Bioinformatics/Phylogenetics%20Assignment/Pinnipeds_initdata.txt), compilation of other species' FASTA DNA sequences from [new_msa_data.txt](https://github.com/itsmichelletrinh/R/blob/main/Bioinformatics/Phylogenetics%20Assignment/new_msa_data.txt), and the associated output from [new_msa_output.txt](https://github.com/itsmichelletrinh/R/blob/main/Bioinformatics/Phylogenetics%20Assignment/new_msa_output.txt).
- Tools: R Studio for data analysis and visualizations - View [R file](https://github.com/itsmichelletrinh/R/blob/main/Bioinformatics/Phylogenetics%20Assignment/BIOL%20365%20Lab%20Assignment%20%233.R).
- Final Report: [BIOL 365 Lab Assignment #3](https://github.com/itsmichelletrinh/R/blob/main/Bioinformatics/Phylogenetics%20Assignment/BIOL%20365%20Lab%20Assignment%20%233.pdf).
